Queremos criar, na forma de código Java, a abstração de uma casa. 
Sabemos que essa possui quatro paredes (Norte, Sul, Leste e Oeste) que possuem largura,
 altura, são feitas de determinado material e podem, ou não, ter acabamento. 
 As paredes também podem ter, ou não, portas e janelas. 
 
 Tanto as portas, quanto as janelas, possuem largura, altura e um estado 
 (que serve para indicar se estão abertas ou fechadas).
 
 A construção da casa é finalizada com um telhado, que é feito de determinado material e 
 cobre toda a área construída da casa.

O código foi desenvolvido no encontro da aula do dia 30/6 e está disponível no Moodle. Então a tarefa é organizá-lo na forma de um projeto do BlueJ.

a) Crie um projeto no BlueJ chamado casaXXXXX, onde XXXX é o seu nome.

b) Teste usando a interface do BlueJ.

c) Envie seu projeto do BlueJ compactado (para isso use a opção Create Jar File do menu Project)
