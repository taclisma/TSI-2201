OBJETIVO DO PROJETO:
Deseja-se desenvolver um sistema para gerenciamento de aluguéis de uma imobiliária.
Os imóveis anunciados são gerenciados por um corretor, que possui um número de
identificação junto ao Conselho Regional de Corretores de Imóveis (CRECI) e que recebe
uma porcentagem do valor do primeiro aluguel como taxa de corretagem. Nesse
anúncio constam as informações de quem é o corretor responsável pelo imóvel, o
proprietário, endereço do imóvel, valor do aluguel, a data em que esse imóvel foi
anunciado e um código de registro.
Quando o aluguel do imóvel é confirmado, o mesmo passa de “livre” para “ocupado”. É
registrado o valor acertado pelo aluguel mensal, a data em que o aluguel deve ser pago,
o valor que o cliente deverá pagar como seguro fiança, o vencimento do contrato e o
número do contrato.
